# 屏幕录制工具

一个功能强大的Python桌面屏幕录制工具，支持录制、标记进度和视频剪辑。

## 功能特点

- 屏幕录制（开始/暂停/停止）
- 可调整录制区域
- 录制时标记进度
- 视频预览和播放
- 可拖拽的进度条
- 视频剪辑功能（剪映风格标记）
- 标记点跳转和重命名

## 使用方法

### 运行程序

```bash
cd d:\代码存档\zbtip
python screen_recorder.py
```

### 录制屏幕

1. 设置录制区域坐标
2. 点击"开始录屏"
3. 需要时点击"暂停录屏"
4. 录制过程中可以点击"标记进度"
5. 点击"结束录屏"完成录制

### 编辑标记

- 点击黄色标记可以跳转到对应位置
- 可以编辑标记名称和备注

### 剪辑视频

1. 点击"截取视频"按钮
2. 拖动绿色开始标记和红色结束标记
3. 调整好位置后点击"完成截取"
